Galactic binaries

Please return Galactic-binary parameter data as ASCII files, where each row describes a source and contains the following elements:

source-index (integer) frequency (Hz) ecliptic latitude (rad) ecliptic longitude (ra) "h" amplitude (adim.) polarization angle (rad) inclination angle (rad) initial phase (rad)

separate elements with spaces or tabs; use the standard scientific notation (x.xxxE-xx) for numbers, except for source-index, which should be an integer. Use "NaN" for parameters that are not returned.

If you are providing confidence intervals, upload them as a separate file in the same encoding, but with two numbers for each parameters:

source-index (integer) min. frequency (Hz) max. frequency (Hz) min. ecliptic latitude (rad) max. ecliptic latitude (rad) ...and so on...

Don't forget the source indexes, which should match those in the corresponding parameter file!
